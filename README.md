# AI Resume ATS Analyzer

An AI-powered resume screening tool that compares resumes with job descriptions
and gives a detailed ATS match report.

## Features
- Works for ANY job role (no hardcoded keywords)
- Section-wise analysis: Skills, Experience, Education, Projects, Certifications
- Matched and missing keyword lists
- Predicted score after improvements
- Downloadable PDF report

## Tech Stack
Python, Streamlit, scikit-learn, TF-IDF, Cosine Similarity, PyPDF2, NLTK, fpdf2

## How to Run
pip install -r requirements.txt
streamlit run app.py
